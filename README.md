# 🚀 Portfolio (Next.js)

This is a modern, fully responsive **personal portfolio website** built using **Next.js 15, React 19, Tailwind CSS, and Radix UI components**. The project showcases skills, projects, and professional information with a clean UI, smooth interactions, and scalable architecture.

---

## 📌 Project Overview

This portfolio is designed to present personal and professional details in an interactive and visually appealing way. It leverages the power of **Next.js App Router**, **component-based architecture**, and **modern UI libraries** to deliver high performance and a seamless user experience.

The application follows best practices such as reusable components, modular structure, and optimized rendering.

---

## 🛠️ Tech Stack

- **Frontend Framework:** Next.js (v15)
- **Library:** React (v19)
- **Styling:** Tailwind CSS + tailwind-merge + tailwindcss-animate
- **UI Components:** Radix UI
- **Icons:** Lucide React
- **Form Handling:** React Hook Form + Zod
- **State & Utilities:** clsx, class-variance-authority
- **Charts & UI Enhancements:** Recharts, Embla Carousel
- **Theme Support:** next-themes
- **Language:** TypeScript

---

## ⚙️ Features

- ✨ Modern and responsive UI
- 🎨 Clean component-based design
- 🌙 Dark/Light theme support
- 📊 Interactive UI elements and charts
- 📁 Organized project structure
- ⚡ Fast performance using Next.js SSR/SSG

## 📂 Project Structure
